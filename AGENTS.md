# RateKit Agent Guide

- This project uses Next.js App Router, TypeScript, Tailwind CSS, and the `src/` directory.
- Keep changes small and focused.
- Prefer editing existing files before adding new libraries.
- Do not add authentication, database, payments, or AI unless explicitly requested.
- Do not modify `package.json` unless the task requires it.
- The main homepage is `src/app/page.tsx`.
